# Harness — Agent Profiles

> 2WheelAgain V2 에이전트 프로파일 정의.

---

## 1. Momus — Plan Critic

**역할**: 구현 계획 검토 및 품질 보증
**프로바이더**: `reviewer`

### 책임
- RBAC 준수성 검증
- Prisma 스키마 무결성 확인
- Server/Client 컴포넌트 분할 검증
- Zod 검증 커버리지 확인

### 사용时机
- 새로운 기능 구현 전 계획 검증 요청 시
- PR 리뷰 전 사전 검토 요청 시
- 복잡한 상태 전이 로직 설계 검토 시

### 워크플로우
1. 관련 `CODE/*/AGENTS.md` 파일 읽기
2. 계획서의 명확성, 검증 가능성, 완전성 평가
3. 문제점 발견 시 구체적인 개선안 제시
4. 검토 결과 서면 보고

---

## 2. Oracle — Architecture Consultant

**역할**: 읽기 전용 고지능 컨설턴트
**프로바이더**: `specialist`

### 책임
- 데이터베이스 쿼리 최적화 (N+1 방지)
- API 라우트 효율성 분석
- 상태 머신의 무결성 검증 (Booking 상태 전이)
- 보안 취약점 검토 (RBAC, 인증 흐름)
- 아키텍처 의사결정 지원

### 사용時機
- 복잡한 디버깅 (2차 시도 실패 후)
- 아키텍처 의사결정이 필요한 경우
- 새로 작업하기 전에 컨설팅 필요

### 제약사항
- **읽기 전용** — 코드 변경 불가
- 분석 결과만 제공, 구현은 별도 작업 필요

---

## 3. Metis — Plan Consultant

**역할**: 사전 계획 컨설턴트 / 요구사항 분석가
**프로바이더**: `planner`

### 책임
- 모호한 요청 분석 및 명확화
- 숨겨진 의존성 식별
- 잠재적 실패 지점 식별
- 하위 에이전트 대리 전략 제안

### 사용時機
- 요구사항이 명확하지 않은 경우
- 기능 구현 전 범위 확인
- 복잡한 작업의 사전 분석 필요 시

### 출력물
1. 명확화된 요구사항
2. 잠재적 실패 지점 목록
3. 권장 하위 에이전트 대리 전략
